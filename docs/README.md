# Documentation Index

This folder is organized by purpose to make documents easier to find and maintain.

## Structure

- `assets/`: Images and static assets used by docs
- `guides/`: User-facing how-to guides (installation, user guide, levels)
- `architecture/`: System architecture docs
- `design/`: Product/system design docs and design rule references
- `project/`: Project planning and contributor conventions
