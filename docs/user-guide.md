## 1. 歡迎

**Rexiano**（Rex + Piano）是一套免費、開源的鋼琴練習軟體，靈感來自 Synthesia，但完全免費且持續進化中。

這個專案最初是一位爸爸為了六歲的兒子 Rex 而打造的——希望讓學琴變得像打電玩一樣有趣。後來我們把它開源出來，希望每一個正在學琴的孩子和大人，都能享受音樂帶來的快樂。

### Rexiano 能做什麼？

- **下落音符顯示** — 像節奏遊戲一樣，音符從螢幕上方落下，你只要跟著彈
- **連接你的鍵盤** — 支援 USB 和藍牙 MIDI 鍵盤（如 Roland、Yamaha 等）
- **三種練習模式** — 觀看、等待、自由，滿足不同階段的練習需求
- **即時評分** — 彈對了會發光，連續答對會出現連擊特效
- **18 首內建曲目** — 從〈小星星〉到〈月光奏鳴曲〉，由易到難
- **匯入你自己的 MIDI 檔案** — 想練什麼就練什麼
- **4 套精美主題** — Lavender、Ocean、Peach、Midnight，總有一套合你口味
- **完全免費** — 沒有廣告、沒有付費牆、沒有訂閱制

### 這份手冊適合誰？

不管你是正在幫孩子找練琴工具的家長，還是自己想學琴的大人，這份手冊都會帶你一步一步上手 Rexiano。不需要任何技術背景，照著做就好。

## 2. 安裝指南

### 2.1 Windows 安裝

1. 前往 [Rexiano 的 GitHub Releases 頁面](https://github.com/nickhsu-endea/Rexiano/releases) 下載最新版本
2. 下載 `rexiano-x.x.x-setup.exe` 檔案
3. 雙擊執行安裝程式
4. 如果出現 **Windows SmartScreen** 警告（「Windows 已保護您的電腦」），請點選「更多資訊」，然後點選「仍要執行」。這是因為 Rexiano 目前尚未購買程式碼簽章憑證，並非惡意軟體
5. 按照安裝精靈的指引完成安裝
6. 安裝完成後，桌面會出現 Rexiano 的捷徑

### 2.2 macOS 安裝

1. 前往 [Rexiano 的 GitHub Releases 頁面](https://github.com/nickhsu-endea/Rexiano/releases) 下載最新版本
2. 下載 `rexiano-x.x.x.dmg` 檔案
3. 雙擊打開 DMG 檔案
4. 將 Rexiano 圖示拖放到「應用程式」資料夾
5. 首次開啟時，macOS 可能會顯示「無法驗證開發者」的警告
   - 前往「系統設定」>「隱私權與安全性」
   - 在底部找到關於 Rexiano 的提示，點選「仍然開啟」
6. 完成！你可以在 Launchpad 或應用程式資料夾中找到 Rexiano

### 2.3 Linux 安裝

Rexiano 提供三種 Linux 安裝格式：

**AppImage（推薦，免安裝）：**
1. 下載 `rexiano-x.x.x.AppImage`
2. 賦予執行權限：`chmod +x rexiano-x.x.x.AppImage`
3. 雙擊執行，或在終端機中執行 `./rexiano-x.x.x.AppImage`

**Debian / Ubuntu（.deb）：**
1. 下載 `rexiano-x.x.x.deb`
2. 使用 dpkg 安裝：`sudo dpkg -i rexiano-x.x.x.deb`
3. 安裝完成後在應用程式選單中找到 Rexiano

**Snap：**
1. 下載 `rexiano-x.x.x.snap`
2. 使用 snap 安裝：`sudo snap install rexiano-x.x.x.snap --dangerous`

### 2.4 首次啟動注意事項

- **新手引導**：第一次啟動 Rexiano 時，會出現一個 4 步驟的引導教學，帶你快速了解基本操作。跟著走就好，大約 1 分鐘即可完成
- **音頻載入**：Rexiano 內建鋼琴音色（SoundFont），首次播放歌曲時會花幾秒鐘載入，之後就會很順暢了
- **MIDI 鍵盤**：如果你有 MIDI 鍵盤，可以在開始前就接上（USB 直接插就好）。沒有鍵盤也完全可以使用 Rexiano，只是不能互動練習

## 3. 快速開始

5 分鐘即可上手！按照以下步驟開始你的第一次練習。

### 步驟 1：選擇一首歌

啟動 Rexiano 後，你會看到**曲庫**畫面。這裡列出了 18 首內建曲目，依照難度分為三級：

| 難度 | 適合誰 | 範例曲目 |
|------|--------|----------|
| Beginner | 剛開始學琴的你 | 小星星、瑪莉有隻小綿羊、生日快樂 |
| Intermediate | 彈了一段時間的你 | 快樂頌、乘著歌聲的翅膀、G 大調小步舞曲 |
| Advanced | 想挑戰自己的你 | 月光奏鳴曲、乾德行進曲 |

直接點選任何一首歌的卡片即可開始。

> **小提示**：如果你有自己的 MIDI 檔案，可以點選右上角的「Open File」按鈕匯入，或者直接把 `.mid` 檔案拖放到 Rexiano 視窗中。

### 步驟 2：按 Space 播放

選好歌後，你會看到下落音符畫面。畫面中間有一條水平線（hit line），音符會從上方落下穿過這條線。

按下鍵盤上的 **Space**（空白鍵）開始播放。音符會隨著音樂落下，你可以先看看、聽聽，感受一下節奏。

再按一次 **Space** 暫停。

### 步驟 3：選擇練習模式

在畫面下方的練習工具列中，你可以切換三種練習模式：

- **Watch**（觀看）— 只看不練，音樂自動播放。適合先熟悉曲目
- **Wait**（等待）— 音樂會在每個音符處暫停，等你彈對了才繼續。這是最推薦的練習方式！
- **Free**（自由）— 音樂持續播放，你可以自由跟彈，系統會記錄你的命中率

對於初學者，我們建議：先用 **Watch** 模式聽一遍，再切到 **Wait** 模式慢慢練。

### 就這樣！

你已經學會了 Rexiano 最核心的操作。接下來的章節會介紹更多進階功能，但現在你已經可以開始練琴了。享受音樂吧！

## 4. 功能詳解

### 4.1 下落音符顯示

Rexiano 的核心畫面是**下落音符顯示**（Falling Notes）。它的運作方式就像一個倒過來的鋼琴卷簾：

```
     時間流動方向
         |
         v
  ┌──────────────────────────────┐
  │                              │  未來的音符
  │    ┌──┐     ┌────┐          │
  │    │  │     │    │          │  音符矩形
  │    └──┘     └────┘          │  （不同顏色代表不同聲部）
  │                              │
  │ ───── hit line ───────────── │  <-- 當前播放位置
  ├──────────────────────────────┤
  │  琴鍵 (88 鍵鋼琴佈局)       │  <-- 音符到達這裡時對應的琴鍵會亮起
  └──────────────────────────────┘
```

- **音符矩形**的水平位置對應琴鍵，垂直長度代表音符的持續時間
- 不同聲部（track）用不同顏色區分，通常 Track 1 是右手、Track 2 是左手
- 音符上會顯示音名標籤（如 C4、F#5），幫助你辨識音符
- 琴鍵上也會顯示音名（可在設定中關閉）

### 4.2 播放控制（TransportBar）

TransportBar 位於畫面下方，提供以下控制：

| 元素 | 說明 |
|------|------|
| 播放/暫停按鈕 | 開始或暫停播放（或按 Space） |
| 時間顯示 | 顯示目前播放位置 / 總時長 |
| 進度滑桿（Seek Bar） | 拖曳可跳轉到任意位置 |
| 音量控制 | 調整播放音量 |
| 節拍器按鈕 | 開啟/關閉節拍器（計時器圖示） |
| 音頻載入狀態 | 載入中會顯示旋轉圖示，錯誤時會顯示警告圖示 |

### 4.3 練習模式

Rexiano 提供三種練習模式，你可以在練習工具列中切換：

#### Watch 模式（觀看）

音樂會自動播放，你只需要看和聽。適合：
- 第一次聽一首新歌
- 觀察左右手的音符分佈
- 純粹享受音樂

#### Wait 模式（等待） -- 最推薦！

這是 Rexiano 最核心的練習功能。在這個模式下：

1. 音樂正常播放
2. 當下一個需要你彈奏的音符到達 hit line 時，**音樂自動暫停**
3. 你在鍵盤上彈出**正確的音**，音樂才會繼續
4. 如果是一個和弦（多個音同時），你需要把**所有音都按下**才算通過
5. 有 200 毫秒的容許時間窗口，不需要分秒不差

這個模式讓你可以按照自己的速度練習——不會被音樂帶著跑，彈錯也不怕，慢慢來就好。

#### Free 模式（自由）

音樂持續播放，不會暫停等你。你跟著彈，系統會即時判定你彈的音是否正確：
- 彈對了：音符會短暫發光
- 彈錯了：音符會變成灰色

適合已經比較熟悉曲目、想要測試自己反應速度的時候使用。

### 4.4 速度控制

練習新曲子時，原速往往太快。Rexiano 讓你可以調整播放速度：

- 速度範圍：**0.25x**（四分之一速）到 **2.0x**（兩倍速）
- 預設按鈕：25% / 50% / 75% / 100%，一鍵切換常用速度
- 連續滑桿：精確調整到任何速度
- 也可以用鍵盤快捷鍵 **上/下方向鍵** 以 25% 為單位微調

建議從 50% 開始練習一首新曲子，熟練後再逐步加速到原速。

### 4.5 A-B 段落循環

遇到一段特別難的旋律？用 A-B 循環功能反覆練習那一小段：

1. 在練習工具列中找到 **A-B Loop** 控制
2. 按下 **A** 按鈕設定循環起點
3. 按下 **B** 按鈕設定循環終點
4. 音樂播放到 B 點時會自動跳回 A 點，不斷循環
5. 在進度滑桿上會看到一段彩色高亮區域，標示循環範圍
6. 按 **L** 鍵或再次點選可以取消循環

### 4.6 分手練習

很多鋼琴曲有兩個聲部：右手旋律和左手伴奏。Rexiano 讓你可以選擇只練某一隻手：

1. 在練習工具列中找到 **Track** 選擇器
2. 勾選你想練習的聲部（track）
3. 未選中的聲部仍然會播放音頻（作為伴奏），但不會要求你彈奏
4. 評分也只計算你選中的聲部

典型練法：先練右手 -> 再練左手 -> 最後合手。

### 4.7 評分系統與慶祝效果

在 Wait 模式和 Free 模式下，Rexiano 會追蹤你的表現：

**即時回饋**：
- 畫面右上角會顯示**即時分數面板**（ScoreOverlay），包含準確率和連擊數
- 彈對一個音符，它會**短暫發光**
- 彈錯或錯過，音符會**變成灰色**
- 連續答對達到里程碑時，會出現**連擊特效**

**練習結算**：
當一首歌練完（或你停止播放）後，會根據準確率顯示結算畫面：

| 準確率 | 效果 |
|--------|------|
| 90% 以上 | 全屏慶祝動畫 + 「太棒了！」 |
| 70%~89% | 星星特效 + 「做得好！」 |
| 70% 以下 | 鼓勵文字 + 「再試一次？」 |

每次練習的成績都會自動保存，你可以在曲庫中看到每首歌的最佳成績徽章：
- 金色徽章：準確率 90% 以上
- 銀色徽章：準確率 70%~89%
- 綠色徽章：準確率 70% 以下

如果打破了歷史最佳紀錄，結算畫面還會出現「新紀錄！」的特別提示。

## 5. 連接 MIDI 鍵盤

Rexiano 支援連接外部 MIDI 鍵盤，讓你在真正的琴鍵上練習。沒有 MIDI 鍵盤也可以使用 Rexiano 的觀看功能，但要使用 Wait 模式和評分功能，你需要一個 MIDI 鍵盤。

### 5.1 USB MIDI 鍵盤（即插即用）

最簡單的連接方式：

1. 用 USB 線將 MIDI 鍵盤連接到電腦
2. 開啟 Rexiano
3. 在歌曲頁面上方找到 MIDI 裝置圖示
4. 點選裝置選擇器，你的鍵盤應該已經出現在列表中
5. 選擇你的鍵盤作為輸入裝置
6. 連線狀態燈會從灰色變成**綠色**，表示連接成功
7. 在鍵盤上隨便按幾個鍵，螢幕上的琴鍵應該會跟著亮起

> **小提示**：如果你的鍵盤有內建喇叭（如 Roland 數位鋼琴），你也可以在裝置選擇器中設定輸出裝置，讓 Rexiano 把音樂透過你鍵盤的喇叭播放出來。

### 5.2 藍牙 MIDI 鍵盤

藍牙 MIDI 鍵盤的設定因作業系統而異：

#### macOS

macOS 原生支援藍牙 MIDI，設定最簡單：

1. 開啟鍵盤的藍牙功能
2. 在 macOS 系統的「藍牙」設定中配對你的鍵盤
3. 配對成功後，回到 Rexiano
4. 你的藍牙鍵盤會自動出現在 MIDI 裝置列表中
5. 選擇它即可開始使用

你也可以在 Rexiano 中直接使用「Bluetooth」按鈕掃描附近的藍牙 MIDI 裝置。

#### Windows

Windows 對藍牙 MIDI 的原生支援較弱，需要額外安裝一個免費橋接軟體：

1. 在 Microsoft Store 中搜尋並安裝 [**MIDIberry**](https://www.microsoft.com/store/apps/9n39720h2m05)（免費）
2. 開啟鍵盤的藍牙功能
3. 在 Windows 設定中配對你的藍牙鍵盤
4. 開啟 MIDIberry，它會自動偵測到你的藍牙 MIDI 裝置並建立虛擬 MIDI Port
5. 回到 Rexiano，在裝置列表中選擇 MIDIberry 創建的虛擬端口
6. 完成！

> **為什麼需要 MIDIberry？** Windows 傳統的 MIDI API 不原生支援藍牙 MIDI 裝置，MIDIberry 作為橋接軟體，將藍牙 MIDI 訊號轉換為標準 MIDI 端口，讓 Rexiano 可以正常讀取。

#### Linux

1. 使用 BlueZ 配對你的藍牙 MIDI 鍵盤：
   ```
   bluetoothctl
   scan on
   pair [設備地址]
   connect [設備地址]
   ```
2. 配對成功後，鍵盤會出現為 ALSA MIDI 裝置
3. 在 Rexiano 中選擇對應的 MIDI 端口

### 5.3 測試連接

不確定鍵盤有沒有接好？使用裝置選擇器中的 **Test** 按鈕：

1. 選擇好輸入裝置後，找到 Test 按鈕
2. 點選後，Rexiano 會進入測試模式
3. 在鍵盤上隨便彈幾個音
4. 如果畫面上的琴鍵有反應（亮起），表示連接正常
5. 測試完成後會自動結束

### 5.4 延遲補償設定

藍牙連接可能會有一點點延遲。如果你覺得在 Wait 模式下判定不太準，可以調整延遲補償：

1. 開啟設定面板（齒輪圖示）
2. 找到「延遲補償」滑桿
3. 從 0ms 開始，逐步增加（建議 10-30ms），直到感覺判定更準確
4. USB 連接通常不需要補償（保持 0ms）

### 5.5 連線狀態指示

裝置選擇器旁邊有一個小圓點，用顏色顯示連線狀態：

| 顏色 | 意義 |
|------|------|
| 綠色 | 已連接，一切正常 |
| 灰色 | 未連接任何裝置 |
| 紅色 | 連線錯誤（請檢查線路或重新插拔） |

## 6. 設定面板

點選歌曲頁面右上角的**齒輪圖示**即可開啟設定面板。

### 6.1 顯示設定

| 設定項目 | 說明 | 預設值 |
|----------|------|--------|
| 琴鍵音名 | 在鍵盤的白鍵上顯示 C、D、E 等音名，C 鍵會標示八度數字（如 C4） | 開啟 |
| 音符音名 | 在下落音符矩形上顯示音名（如 C4、F#5） | 開啟 |

### 6.2 音頻設定

| 設定項目 | 說明 | 預設值 |
|----------|------|--------|
| 音量 | 主音量，0~100% | 80% |
| 靜音 | 一鍵靜音（快捷鍵 M） | 關閉 |

### 6.3 練習預設

| 設定項目 | 說明 | 預設值 |
|----------|------|--------|
| 預設速度 | 每次開始新練習時的初始速度 | 100% (1.0x) |
| 預設模式 | 每次開始新練習時的初始模式 | Watch |

### 6.4 節拍器設定

| 設定項目 | 說明 | 預設值 |
|----------|------|--------|
| 節拍器 | 開啟/關閉節拍器 | 關閉 |
| 預備拍數 | 按下播放後，在正式開始前先數幾拍 | 4 拍 |

### 6.5 延遲補償

| 設定項目 | 說明 | 預設值 |
|----------|------|--------|
| 延遲補償 | 補償 MIDI 鍵盤的輸入延遲（0~100ms） | 0ms |

### 6.6 主題切換

Rexiano 提供四套精心設計的視覺主題：

| 主題 | 風格 | 建議場景 |
|------|------|----------|

## 8. 常見問題 FAQ

### Q: 聽不到聲音怎麼辦？

1. **檢查音量**：確認 Rexiano 的音量滑桿不是 0%，也確認沒有開靜音（按 M 鍵切換）
2. **檢查系統音量**：確認你電腦的系統音量沒有靜音
3. **等待載入**：首次播放歌曲時，SoundFont 音色檔案需要幾秒鐘載入。如果 TransportBar 上顯示旋轉的載入圖示，請耐心等待
4. **音頻載入失敗**：如果 TransportBar 顯示紅色警告圖示，表示音色載入失敗。請嘗試重新啟動 Rexiano
5. **瀏覽器限制**：現代瀏覽器（包括 Electron）需要使用者互動後才會啟動音頻引擎。確保你有點選過播放按鈕或按過 Space 鍵

### Q: 鍵盤連不上怎麼辦？

1. **USB 鍵盤**：
   - 拔掉 USB 線，等 3 秒，重新插上
   - 嘗試換一個 USB 端口
   - 確認鍵盤的電源已開啟
   - 在 Rexiano 的裝置選擇器中重新選擇你的鍵盤

2. **藍牙鍵盤**：
   - 確認鍵盤的藍牙功能已開啟
   - 確認已在作業系統中完成藍牙配對
   - Windows 使用者：確認 MIDIberry 正在運行
   - 嘗試取消配對後重新配對

3. **通用排查**：
   - 重新啟動 Rexiano
   - 確認沒有其他程式佔用了 MIDI 裝置（如 DAW 軟體）

### Q: 如何匯入自己的 MIDI 檔案？

三種方式：

1. **拖放匯入**（最簡單）：直接把 `.mid` 或 `.midi` 檔案拖進 Rexiano 視窗中
2. **檔案對話框**：按 `Ctrl+O`（Windows/Linux）或 `Cmd+O`（macOS），在彈出的對話框中選擇檔案
3. **曲庫中的「Open File」按鈕**：在曲庫頁面點選右上角的按鈕

MIDI 檔案可以從以下地方取得：
- [MuseScore](https://musescore.com/) — 大量免費樂譜，可匯出 MIDI
- [IMSLP](https://imslp.org/) — 公版古典樂 MIDI
- 任何 MIDI 編輯軟體（如 GarageBand、FL Studio）匯出的檔案

### Q: Windows SmartScreen 警告如何處理？

Rexiano 目前尚未購買 Windows 的程式碼簽章憑證（EV Code Signing Certificate），所以 Windows Defender SmartScreen 可能會在你首次安裝時跳出警告。

**這不代表 Rexiano 是惡意軟體。**

處理方式：
1. 看到「Windows 已保護您的電腦」警告時，點選「更多資訊」
2. 點選「仍要執行」
3. 繼續安裝即可

Rexiano 是完全開源的軟體，你可以在 [GitHub](https://github.com/nickhsu-endea/Rexiano) 上查看所有原始碼。

### Q: 最近開啟的檔案在哪裡？

在曲庫頁面的頂部，你會看到「最近」區域，列出你最近開啟過的 10 個 MIDI 檔案。直接點選就能快速重新開啟，不需要再去檔案對話框找。

### Q: 為什麼有些音符上沒有顯示音名？

如果一個音符的持續時間太短（矩形高度小於 16 像素），Rexiano 會自動隱藏音名標籤以避免文字溢出重疊。這是正常的，通常出現在非常快速的裝飾音或三十二分音符上。

### Q: 支援哪些 MIDI 鍵盤？

Rexiano 支援任何符合 MIDI 標準的鍵盤，包括但不限於：
- Roland（FP-30X、FP-60X、GO:KEYS 等）
- Yamaha（P-125、CLP 系列等）
- Casio（PX 系列、CDP 系列等）
- Korg、M-Audio、Arturia 等品牌
- 任何 USB 或藍牙 MIDI 控制器

## 9. 給家長的建議

親愛的爸爸媽媽們，Rexiano 的誕生就是一位爸爸對孩子學琴路上的愛與陪伴。以下是一些讓練琴更快樂、更有效的小建議。

### 每天 15-20 分鐘就好

對於 6-10 歲的孩子來說，每天 15-20 分鐘的專注練習，比一口氣練 1 小時要有效得多。短時間的練習更容易養成習慣，孩子也不會覺得是負擔。

**小技巧**：可以把練琴時間安排在每天固定的時段（例如晚餐前或寫完功課後），讓它變成日常的一部分。

### 從簡單的歌開始

Rexiano 內建的歌曲按照難度分級。一定要從 **Beginner** 難度的歌開始：

- 《小星星》（Twinkle Twinkle） — 最經典的入門曲
- 《瑪莉有隻小綿羊》（Mary Had a Little Lamb） — 只用幾個音
- 《Hot Cross Buns》 — 三個音就能彈的歌

先讓孩子在簡單的歌中建立成就感，再慢慢挑戰更難的曲目。

### 善用 Wait 模式 + 慢速

最推薦的練習組合：

1. 切到 **Wait 模式**（按 2 鍵）
2. 把速度調到 **50%**（按幾次方向鍵下）
3. 讓孩子按照自己的節奏，一個音一個音慢慢彈

Wait 模式不會讓孩子跟不上而感到挫折，慢速則給了足夠的反應時間。等到一段旋律彈熟了，再慢慢加速。

### 慶祝進步，不要苛求完美

Rexiano 的評分系統是為了鼓勵，不是為了給壓力。

- 準確率從 40% 進步到 60%？太棒了，這就是進步！
- 這次比上次多連擊了 5 個音符？值得慶祝！
- 第一次把一首歌完整彈完？這是里程碑！

每個孩子的學習速度不同，重要的是享受過程，而不是追求分數。當孩子看到自己的成績徽章從綠色變成銀色、再變成金色，那份成就感會成為持續練習的最大動力。

### 一起玩！

有時候，和孩子一起坐在鋼琴前，用 Watch 模式看音符落下，聊聊這首歌好不好聽，一起拍手跟節奏——這比任何練習技巧都重要。

音樂最美好的地方，是它能連結人。

祝你和孩子在 Rexiano 中找到練琴的快樂！
